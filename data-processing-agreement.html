Data Processing Agreement (DPA)

Last Updated: April 8, 2026

1. Data Controller

The App developer acts as the data controller for authentication and subscription-related data.

2. Data Storage

All user-generated content (tasks, flocks, productivity data) is stored locally on the user's device.

3. Third-Party Processors

The App uses the following third-party services:

  • Firebase Authentication (Google LLC) – for user login
  • RevenueCat – for subscription management

4. No Server-Side Data Storage

The App does not store user productivity data on external servers.

5. User Rights

Users may delete their data at any time by deleting the application from their device.
